Pengantar HTML

Penghantar HTML adalah elemen atau tag yang digunakan untuk menentukan bagaimana sebuah konten atau struktur harus ditampilkan dalam sebuah halaman web. Ini termasuk tag-tag seperti table, div, p, header, footer, dan banyak lagi

Elemen Semantik HTML

Elemen semantik adalah elemen dalam HTML yang memiliki makna atau arti tertentu yang jelas, dan bukan hanya digunakan untuk tujuan penataan atau penampilan visual. Dengan kata lain, elemen semantik memberikan informasi tentang struktur dan makna konten, sehingga membantu mesin pencari, pembaca layar, dan pengembang web untuk memahami dan menafsirkan konten dengan lebih baik

Tutorial Membuat Layout Sederhana

Ikuti langkah langkah mudha di bawah ini!

  • Mulai dengan deklarasi tipe dokumen HTML ()
  • Tentukan bahasa dokumen HTML dan atur tampilan awal untuk perangkat mobile
  • Gunakan tag head untuk menyertakan informasi meta dan tautan ke berkas CSS dan JavaScript
  • Gunakan tag title untuk menentukan judul halaman
  • Gunakan tag style untuk menambahkan aturan CSS
  • Gunakan tag body untuk menempatkan konten utama halama
  • Gunakan tag body untuk menempatkan konten utama halaman
  • Gunakan elemen HTML seperti header, nav, main, aside, dan footer untuk mengatur struktur layout.
  • Gunakan tabel atau tata letak CSS fleksibel untuk menyusun elemen-elemen tersebut
  • Sesuaikan tampilan dengan media queries agar responsif di berbagai perangkat
  • Tutup dokumen HTML dengan tag html

Cara Mudah Mengatur Layout di HTML

  • Gunakan Struktur Dasar HTML: Mulailah dengan menentukan struktur dasar HTML dengan menggunakan tag html, head, dan body
  • Gunakan Tag Semantik: Gunakan tag-tag semantik HTML seperti header, nav, main, section, article, aside, dan footer untuk membagi halaman web menjadi bagian-bagian yang terstruktur
  • Gunakan CSS untuk Tata Letak (Layout): Gunakan CSS untuk mengatur tata letak (layout) halaman web, seperti mengatur lebar, tinggi, margin, padding, dan posisi elemen-elemen HTML.
  • Tambahkan Konten: Terakhir, tambahkan konten sesuai dengan struktur yang telah ditentukan
